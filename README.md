# Project ZsorMC
An open source website for the minecraft server called ZsorMC. This project made with VueJS.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
